# CFRK Package

An R package for spatial analysis using Compositional Fixed Rank Kriging. An
example is the estimate of the probability of exceeding the daily threshold for
PM10, as described in De Sanctis et al. (2025).

## Contents

- **CFRK.R** - Main package functions
- **CFRK_utils.R** - Utility functions


## Installation

Load the package in R:

```r
source("CFRK.R")
source("CFRK_utils.R")
```

## Usage

Refer to the example scripts in the `example/` folder to see how to use the
package functions.
